A群（行政群）操作手冊

LINE Bot 貸款案件管理系統

一、訊息格式

姓名 + 公司 + 狀態 + 備註（備註多長都可以）：

王小明 亞太核准12萬 @AI
陳世明 和裕婉拒 備註內容可以很長... @AI
蘇畇云 21補件 @AI
唯一規則：姓名和公司之間一定要有空格
✅ 王小明 亞太核准12萬
❌ 王小明亞太核准12萬（會抓錯名字）
其他地方可以不空格，全形半形都可以。

二、觸發方式

操作要 @AI 嗎
撥款名單❌ 不用
回貼核准/婉拒/補件✅ 要
@AI 指令（查/結案/改名等）✅ 要

三、關鍵字判定

核准（優先）

訊息中有 核准 或 核準 → 判定核准，自動抓金額，移到待撥款。

婉拒（從日報移除，推下一家）

婉拒申覆失敗建議維持原審
不予承作無法再進件無法承作
30日內有進件已建檔不提供申覆

補件類（日報顯示標籤）

訊息關鍵字日報顯示
補照會 / 照會已補照會
補申覆 / 申覆已補申覆
補件 / 補資料 / 補行照 / 補聯徵 / 補保人 / 補薪轉 / 補照片 / 補時段已補資料
NA / 未接照會NA
核准優先規則：同時有「核准」和婉拒關鍵字 → 以核准為準
例：「申覆失敗維持原審 核准 5W」→ 判定為核准

四、公司抓取規則

系統只看狀態關鍵字之前的公司，後面備註忽略：
陳世明 亞太 婉拒 近期銀行核貸(玉山+凱基)... → 抓到亞太
備註提到銀行/玉山/凱基不會誤判。

五、撥款名單（不用 @AI）

4/14 和裕 撥款
王小明
李大華
格式：日期 公司 撥款 + 每行一個姓名
也支援「撥款名單」「排撥」「今日撥款」「撥款」等關鍵字。

七、同時送件（照會 +）

@AI 照會 王小明 亞太+和裕+鄉
用 + 連接多家公司，系統會：
1. 回覆一則照會注意事項
2. 回覆第二則同時送件資訊（自動帶金額期數）
3. 日報每家都顯示該客戶，各自獨立狀態

核准 → 從同送移到待撥款，其他繼續
婉拒 → 從同送移除該家

八、公司不在路線跳按鈕

A 群貼的公司不在客戶路線/同送清單 → 跳按鈕：
[再送XX] [同送XX] [取消]

點再送/同送會有二次確認（防誤點）：
[✅ 確定再送XX] [↩️ 返回]

- 再送 → 加到送件順序，設為當前公司
- 同送 → 加到同時送件清單
- 取消 → 不處理

九、多筆一次貼（空一行）

每筆中間按兩次 Enter 留空一行：

王小明 亞太核准12萬

李大華 和裕婉拒

張美玲 21補件
@AI

十、民間方案處理規則

業務群貼民間方案（銀行/零卡/房地/代書/當舖/鄉民/新鑫等）時，系統不推 A 群，只記錄到日報。
因為民間方案通常業務自己處理，不需要 A 群通知。

A 群貼民間方案 → 照常回貼業務群（跟其他方案一樣）。

十一、銀行/零卡類別名

銀行：中信、台新、國泰、富邦、玉山、永豐、新光、凱基、元大、兆豐、華南、聯邦、遠東、王道、星展、渣打、匯豐、花旗、樂天、合庫、土銀、台銀、彰銀、板信、陽信、京城、安泰、日盛、台企銀等 → 自動歸「銀行」

零卡：大哥付、幫你付、遠信、月付大人、銀角、慢點付、先享後付、PI錢包、FULA、分期趣、刷卡換現、信用卡換現 → 自動歸「零卡」

十二、標記其他人

@AI 和標記其他人可以同一則訊息：
王小明 亞太婉拒 @AI @大幫手 @店長 ✅
@AI @大幫手 王小明 亞太婉拒 ✅（@AI 放哪都可以）
王小明 亞太婉拒 @AI@大幫手@小明 ✅（不用空格）

不管標幾個人、在哪個位置，只要有 @AI 就會觸發。

十五、注意事項

  • 姓名後面一定要空格
  • 看到 ❌ 錯誤訊息 → 截圖回報管理員
  • 多筆一次貼要空一行（按兩次 Enter）
  • 同時送件的客戶，婉拒一家只移除該家，其他繼續
